Análisis de encuestas con R

9.11 Imputación múltiple

La imputación múltiple (Rubin, 1987) aborda el problema de los datos faltantes generando \(M > 1\) conjuntos de valores plausibles para las observaciones no observadas. Cada conjunto corresponde a una realización del modelo de imputación, de modo que la incertidumbre asociada a los valores faltantes queda explícitamente representada en la variabilidad entre imputaciones.

En este enfoque, el objetivo es aproximar la distribución condicional de los datos faltantes dado la información observada, es decir,

\[P(Y_{NoObs} \mid Y_{Obs}, X),\] donde \(Y_{Obs}\) el subconjunto de \(n_1\) valores observados de \(Y\), y \(Y_{NoObs}\) el subconjunto de \(n_0 = n - n_1\) valores no observados. Se asume que la covariable \(X\) es completamente observada.

Para ilustrar el procedimiento, se considera un esquema generador de datos donde las variables X e Y se relacionan mediante el siguiente modelo de regresión sin intercepto:

\[ y_i = \beta x_i + \varepsilon_i, \qquad \varepsilon_i \sim N(0,\sigma^2), \qquad i=1,\ldots,n. \] Este modelo se utiliza únicamente como mecanismo generador de datos para la simulación y la ilustración del método. En el marco de la imputación múltiple, el modelo de imputación tiene carácter auxiliar y busca aproximar la distribución condicional de los datos faltantes dado la información observada, independientemente de que el modelo sustantivo incluya o no intercepto.

En imputación múltiple, el modelo de imputación no tiene por qué coincidir con el modelo de análisis final. Su función es aproximar adecuadamente la distribución de los datos faltantes condicional a la información disponible, de forma que la variabilidad entre imputaciones refleje correctamente la incertidumbre asociada al proceso de imputación.

9.11.1 Estimadores de la imputación múltiple

Para el parámetro de interés \(\beta\), la estimación combinada sobre las \(M\) imputaciones es:

\[\bar{\hat{\beta}} = \frac{1}{M} \sum_{m=1}^{M} \hat{\beta}_m\]

La varianza total estimada combina la varianza dentro de las imputaciones (\(\bar{U}\)) y la varianza entre imputaciones (\(B\)), con una corrección por el número finito de imputaciones:

\[\hat{V}(\hat{\beta} \mid Y_{obs}) = \bar{U} + \left(1 + \frac{1}{M}\right) B\]

donde:

\[\bar{U} = \frac{1}{M} \sum_{m=1}^{M} \widehat{Var}(\hat{\beta}_m)\]

\[B = \frac{1}{M-1} \sum_{m=1}^{M} \left(\hat{\beta}_m - \bar{\hat{\beta}}\right)^2\]

9.11.2 Implementación mediante bootstrap

Para ilustrar la metodología, se simula un conjunto de datos con \(n = 500\) observaciones y \(n_0 = 200\) valores faltantes, con pendiente \(\beta = 10\) y desviación estándar \(\sigma = 2\), presentada en la Tabla 9.38::

generar <- function(n = 500, n_0 = 200, beta = 10, sigma = 2) {
  x       <- runif(n)
  mu      <- beta * x
  y       <- mu + rnorm(n, mean = 0, sd = sigma)
  datos   <- data.frame(x = x, y = y)
  faltantes <- sample(n, n_0)
  datos$faltantes       <- "No"
  datos$faltantes[faltantes] <- "Si"
  datos$y.per           <- y
  datos$y.per[faltantes] <- NA
  return(datos)
}

La función de imputación por bootstrap genera \(M\) conjuntos de valores plausibles y devuelve el estimador combinado de \(\beta\) junto con su error estándar:

im.bootstrap <- function(datos, M = 15) {
  library(dplyr)
  n     <- nrow(datos)
  datos1 <- na.omit(datos)
  n1    <- nrow(datos1)
  n0    <- n - n1
  Ind   <- is.na(datos$y.per)
  beta1 <- sigma1 <- NULL

  for (m in 1:M) {
    datos.m      <- dplyr::sample_n(datos1, n1, replace = TRUE)
    model1       <- lm(y ~ 0 + x, data = datos.m)
    beta         <- model1$coeff
    sigma        <- sqrt(anova(model1)[["Mean Sq"]][2])
    faltantes.bt <- rnorm(n0, datos$x[Ind] * beta, sd = sigma)
    datos$y.per[Ind] <- faltantes.bt
    model.input  <- lm(y.per ~ 0 + x, data = datos)
    beta1[m]     <- model.input$coeff
    sigma1[m]    <- summary(model.input)$coeff[2]
  }

  beta.input <- mean(beta1)
  u.bar      <- mean(sigma1^2)
  B          <- var(beta1)
  beta.sd    <- sqrt(u.bar + B + B / M)
  list(new = datos, beta = beta.input, sd = beta.sd)
}

9.11.3 Imputación múltiple para la encuesta de hogares

A continuación, se aplica la imputación múltiple a la encuesta de hogares. Se generan \(M = 10\) valores plausibles para el ingreso y el empleo, ajustando en cada iteración un modelo lineal y uno multinomial a una muestra bootstrap de los datos observados:

encuesta$Income_imp     <- encuesta$Income_missin
encuesta$Employment_imp <- encuesta$Employment_missin
encuesta_obs    <- filter(encuesta, !is.na(Income_missin))
encuesta_no_obs <- filter(encuesta, is.na(Income_missin))
n0 <- nrow(encuesta_no_obs)
n1 <- nrow(encuesta_obs)
M <- 10
set.seed(1234)
for (ii in 1:M) {
  vp  <- paste0("Income_vp_",     ii)
  vp2 <- paste0("Employment_vp_", ii)

  encuesta_temp <- encuesta_obs %>% sample_n(size = n1, replace = TRUE)

  mod      <- lm(Income ~ Zone + Sex + Expenditure, data = encuesta_temp)
  mod.mult <- multinom(Employment ~ Zone + Sex + Expenditure,
                       data = encuesta_temp, trace = FALSE)

  encuesta_no_obs[[vp]]  <- predict(mod,      encuesta_no_obs)
  encuesta_obs[[vp]]     <- encuesta_obs$Income

  encuesta_no_obs[[vp2]] <- predict(mod.mult, encuesta_no_obs, type = "class")
  encuesta_obs[[vp2]]    <- encuesta_obs$Employment
}
