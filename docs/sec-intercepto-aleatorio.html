Análisis de encuestas con R

8.5 Modelo con intercepto aleatorio

El modelo con intercepto aleatorio permite que el nivel medio del ingreso varíe entre estratos, al tiempo que estima una pendiente común para el efecto del gasto. El modelo es:

\[ Ingreso_{ij} = \beta_{0j} + \beta_1 Gasto_{ij} + \epsilon_{ij} \]

donde el intercepto aleatorio se descompone como:

\[ \beta_{0j} = \gamma_{00} + \gamma_{01} Stratum_j + \tau_{0j} \]

con \(\tau_{0j} \sim N(0, \sigma_\tau^2)\) y \(\epsilon_{ij} \sim N(0, \sigma_\epsilon^2)\). El ajuste en R es:

mod_Int_Aleatorio <- lmer(Income ~ Expenditure + (1 | Stratum),
                          data = encuesta, weights = wk2)
performance::icc(mod_Int_Aleatorio)
## # Intraclass Correlation Coefficient
## 
##     Adjusted ICC: 0.196
##   Unadjusted ICC: 0.102

La Figura 8.7 muestra las rectas de regresión estimadas por estrato, donde se aprecia la variación en el intercepto:

Coef_Estimado <- inner_join(
  coef(mod_Int_Aleatorio)$Stratum %>% tibble::rownames_to_column(var = "Stratum"),
  encuesta_plot %>% dplyr::select(Stratum) %>% distinct()
)
ggplot(data = encuesta_plot,
       aes(y = Income, x = Expenditure, colour = Stratum)) +
  geom_jitter() +
  geom_abline(data = Coef_Estimado,
              mapping = aes(slope = Expenditure,
                            intercept = `(Intercept)`,
                            colour = Stratum)) +
  theme(legend.position = "none",
        plot.title = element_text(hjust = 0.5)) +
  theme_cepal()

Figura 8.7: Rectas de regresión estimadas por estrato: modelo con intercepto aleatorio

La variación en el intercepto entre estratos es considerable, lo que confirma que el modelo con intercepto aleatorio captura diferencias estructurales en el nivel de ingreso entre grupos. Las predicciones del modelo se presentan en la Tabla 8.6:

tab_pred <- data.frame(  Predicción = predict(mod_Int_Aleatorio),
  Ingreso    = encuesta$Income,
  Estrato    = encuesta$Stratum) %>% 
  distinct() %>% slice(1:6L)
